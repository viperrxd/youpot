## 📌 Why YouPOT?

YouTube has increasingly implemented bot protection measures, causing third-party clients and music bots to fail with errors like "Sign in to confirm you're not a bot."

To bypass this, Lavalink's [youtube-source](https://github.com/lavalink-devs/youtube-source) plugin allows you to provide a **Proof of Origin Token (`poToken`)** alongside a **`visitorData`** string. YouPOT (YouTube Proof of Origin Token) acts as an automated, headless browser that constantly fetches these required values and serves them over a simple API.

### Features
- **Headless Chrome Automation:** Uses Puppeteer to mimic real browser behavior and extract authentic tokens directly from YouTube's API traffic.
- **Smart Caching:** Avoids rate limits by caching tokens and refreshing them in the background before they expire.
- **Independent & Lightweight:** Zero reliance on third-party token-generation NPM packages. 
- **Simple JSON API:** Returns clean, easy-to-use JSON responses.

## 🎧 Lavalink Integration

Once you have YouPOT running, you can use the tokens it generates in your Lavalink configuration. 

In your Lavalink `application.yml`, configure the `youtube` plugin to use the `pot` (Proof of Origin Token) section:

```yaml
plugins:
  youtube:
    enabled: true
    allowSearch: true
    allowDirectVideoIds: true
    allowDirectPlaylistIds: true
    # Provide the tokens generated by YouPOT here:
    pot:
      token: "YOUR_GENERATED_poToken_HERE"
      visitorData: "YOUR_GENERATED_visitorData_HERE"
```

*Note: You generally do not need to use both `oauth` and `pot` in the youtube-source plugin. If you're using YouPOT, provide these values and disable OAuth.*

## 🚀 Deployment

YouPOT can be deployed anywhere that supports Docker or Node.js. 

### Docker (Recommended)

You can run the pre-built image directly from the GitHub Container Registry. Create a `compose.yml` file:

```yaml
services:
  youpot:
    image: ghcr.io/viperrxd/youpot:latest
    container_name: youpot
    restart: unless-stopped
    ports:
      - "8080:8080"
```

Start the container:
```bash
docker compose up -d
```

### Direct / Source Deployment

If you are deploying from source to a VPS or cloud host:

1. Clone or fork this repository.
2. Ensure you have **Node.js 18+** installed.
3. Install dependencies: `npm install`
4. Start the server: `npm start` (or `node server.js`)

*Since YouPOT uses Puppeteer, make sure the environment has the necessary system dependencies for Chromium installed (most Docker environments handle this automatically).*

## 📡 API Endpoints

Your server exposes the following endpoints:

| Method | Endpoint       | Description                                  |
|--------|----------------|----------------------------------------------|
| `GET`  | `/api/token`   | Returns the currently cached `poToken` and `visitorData` in JSON format. |
| `GET`  | `/health`      | Shows server uptime, cache status, and browser connection state. |
| `GET`  | `/`            | Basic service information.                   |
